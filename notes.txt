** 47-3 Components and how props works, unidirectional data flow
--
+ https://reactjs.org/docs/components-and-props.html
+ Components are independent and reusable bits of code. They serve the same purpose as JavaScript functions, but work in isolation and return HTML.
+ Components come in two types, Class components and Function components, We will concentrate on Function components.
+ React follow unidirectional data flow. Meaning, the data flows from parent to child.
+ props.price = 1400 // not possible because of, props is read only
+ props drilling = props only follow data flows from parent to child this is call props drilling.

** 47-5 How React Hook works, send state via props , props vs state
--
+ hook is one kind of function. Hooks make React so much better because you have simpler code that implements similar functionalities faster and more effectively.
+ hooks likes, useState, useEffect, useContext, useReducer, useCallback, useMemo, useRef, useImperativeHandle, useLayoutEffect, useDebugValue
+ props read only and can not be modified
+ state changes can be asynchronous and state can be change using this.setState()
+ যেই জায়গায় state থাকে ঐ component কে stateful component বলে
